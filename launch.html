Moth and Money — how to start the app

This folder contains a launcher so you do not have to type the Streamlit command every time.

macOS (recommended)

In Finder, open this project folder and double-click launch.command. A Terminal window will open and start the app. If macOS blocks it the first time, use Right-click → Open and confirm.

If double-click does nothing, open Terminal, cd to this folder, run:
chmod +x launch.command
then try double-clicking again.

Terminal (any OS)

From this project folder (the one that contains the app directory):

source app/.venv/bin/activate
streamlit run app/app.py

Or run ./launch.sh if you use the shell script.

After it starts

Your browser should open to the app (usually http://localhost:8501). Stop the server with Ctrl+C in the Terminal window.
